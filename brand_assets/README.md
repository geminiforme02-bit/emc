# Brand Assets — E=MC² Burgers

Drop real brand files here and they'll be picked up by the site.

## Logo
Instagram profile pictures are behind a login wall with signed, expiring CDN
URLs, so the logo could not be auto-fetched from @emc.burgers.

The site currently uses a hand-built **`E=mc²` SVG wordmark** as the logo
(see the `.brand-logo` markup in `index.html`).

To use the real logo instead:
1. Save the logo here as `logo.png` (or `logo.svg`), ideally transparent PNG.
2. Tell me and I'll swap the SVG wordmark for an `<img src="brand_assets/logo.png">`.
